# Tanishka's Portfolio Website 🌐

A responsive and modern personal portfolio website built to showcase my skills, projects, and contact information. This site serves as an online resume and a central hub for anyone interested in learning more about my work and capabilities.

---

## 📌 Objective

The objective of this project is to create a clean and professional personal portfolio website that includes:

- An **About Me** section
- A showcase of **Skills**
- A list of **Projects** with details and links
- A functional **Contact Form**
- A fully **Responsive Design** compatible across all devices

---

Visit the live site here:  
👉 [https://tanishka-portfolio-blush.vercel.app/](https://tanishka-portfolio-blush.vercel.app/)

## 🛠️ Tech Stack Used

- **Language**: JavaScript (ES6+)
- **Framework**: React.js
- **Styling**: Tailwind CSS
- **Build Tool**: Vite
- **Hosting**: Vercel

## 🧑‍💻 Steps to Run This Project Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/tanishkasharma29/my_portfolio.git
   cd my_portfolio

2. **Install all dependencies**
   ```bash
   npm install

3. **Start the development server**
   ```bash
   npm run dev

4. **Open your browser and go to** http://localhost:5173
